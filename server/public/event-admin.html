🎯 TTCQ — mes thèmes perso

Ajoute tes propres thèmes de questions : chaque thème contient 8 questions (niveaux 1 à 8, la 8 impossible). Format JSON : [ { id, name, cat, level, levels: [{q, a}] } ]. Deux formats de réponse au choix, par question : saisie libre {q, a:"Paris"} (le joueur tape un mot ou un nombre) ou QCM {q, c:["Paris","Lyon","Nice","Metz"], a:"Paris"} (4 propositions ; a = la bonne). Prioritaires sur le catalogue.
