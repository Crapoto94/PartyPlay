# 🎒 Matériel & préparation — PIXEL PANIC

> Tout est en **fournitures de bureau** : pas de cadenas, pas d'achat compliqué. Le « verrouillage » est numérique (code tapé dans l'appli).

## À préparer

### Électronique
- [ ] **1 tablette** = la BORNE (sur un support, chargeur branché, mise en veille désactivée, son activé).
- [ ] **6–7 smartphones** (les invités) — bien chargés / sur le WiFi de la maison.
- [ ] **Ton téléphone** = page GM ouverte en plus.
- [ ] WiFi qui couvre la pièce de jeu.

### Les COLIS (cœur de la distribution progressive)
- [ ] **6 enveloppes/pochettes opaques** numérotées « COLIS 1 » à « COLIS 6 » (kraft A5, ou pochettes plastique opaques + autocollant). Scellées simplement.
- [ ] À l'intérieur de chaque colis : le **matériel du monde** correspondant (cf. ci-dessous).

### Cartes & jetons
- [ ] **6 cartes-avatars** (1 par invité) — imprime la fiche perso, glisse dans une pochette. *(je peux te générer des cartes à imprimer si tu veux)*
- [ ] **1 carte dorée « PLAYER ONE »** dans une enveloppe scellée → remise à Vincent au Monde 4.
- [ ] **PIÈCES** : jetons / pièces chocolat dorées / pièces carton (récompenses).
- [ ] **VIES** : 3 cœurs par joueur (gommettes, petites cartes, ou bracelets).

### Petit matériel d'énigmes (selon monde)
- [ ] **Monde 1** : rien de physique (la séquence est sur les téléphones).
- [ ] **Monde 2** : un **plan de labyrinthe** imprimé + 1 indice « secret » (option : écrit au **stylo encre invisible UV**, ~5 €, avec sa mini-lampe — pour LARA CROÛTE / Marc).
- [ ] **Monde 3** : **rien de physique** — une devinette ouvre une « porte sonore », puis les téléphones alignés forment un **piano géant** ; le groupe rejoue la mélodie affichée par la BORNE.
- [ ] **Monde 4** : la carte dorée + (option) une « manette » déco à remettre à Vincent.
- [ ] **Monde 5** : **LA GRANDE ENQUÊTE** — les pièces à conviction sont distribuées **automatiquement via les téléphones**. Option : imprimer le dossier complet (cartes + plan + solution MJ) via le bouton **🖨️ Imprimer le matériel d'enquête** de la page GM (`/print-enquete`).
- [ ] **Monde 6** : le **vrai cadeau / coffre final** de Vincent (déverrouillé à la victoire).

### Ambiance
- [ ] Quelques boissons **avec et sans alcool** (gages légers — toujours une alternative).
- [ ] De quoi grignoter / le gâteau pour la pause (~2 h 30).
- [ ] Option déco : guirlande, manettes, posters rétro, lumière tamisée.
- [ ] Rôle des enfants **Robin & Juliette** : badges « PIXEL », un panier pour transporter les colis, des bonbons-récompenses.
